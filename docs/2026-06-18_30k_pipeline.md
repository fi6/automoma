## Recording Semantics

Use set-state replay, no interpolation, and absolute mobile-base actions:

```bash
--set_state --interpolated 1 --interpolation_type none
```

Do not pass `--mobile_base_relative`.

HDF5 recording is chunked to limit local/taro working storage. The active
chunk is written under `_recording/`; after a chunk succeeds it is moved to
`ready/`; `archive_hdf5_chunks.py` validates, copies to `$ARCHIVE_ROOT`, validates
again, and deletes the working copy.

Default chunk size for this run: 25 episodes per HDF5.

Each chunk is retried up to 3 times by `run_record_worker.sh`. This protects
against transient startup/network failures such as `lightwheel_sdk` registry
timeouts. A chunk is only marked done after the HDF5 file is produced and moved
to `ready/`.

## Local DP3 Trial Train

On 2026-06-20, while H100 was still unavailable, a local RobotWin DP3 trial job
was launched on the local RTX 3090. The job first converts the completed 30K
archive to a DP3 zarr dataset on the external drive, then starts training from
that zarr.

Output paths:

```text
$ARCHIVE_ROOT/robotwin_dp3_data/microwave_7221-automoma_30k-30000.zarr
$ARCHIVE_ROOT/train/robotwin/dp3_microwave_7221_30k_local_trial
$ARCHIVE_ROOT/logs/local_train/local_dp3_30k_trial_20260620_022635.out
```

The training run intentionally uses short epochs on local hardware:

```bash
training.num_epochs=300
training.checkpoint_every=25
training.val_every=25
training.max_train_steps=100
training.max_val_steps=10
dataloader.batch_size=64
val_dataloader.batch_size=64
dataloader.num_workers=4
val_dataloader.num_workers=4
logging.mode=offline
```

Launch command:

```bash
ARCHIVE_ROOT=/media/xinhai/GIANT/Research/AutoMoMa/archieve/repo-temp/2026-06-18
ZARR="$ARCHIVE_ROOT/robotwin_dp3_data/microwave_7221-automoma_30k-30000.zarr"
TRAIN_OUT="$ARCHIVE_ROOT/train/robotwin/dp3_microwave_7221_30k_local_trial"

cd /home/xinhai/projects/lerobot-arena

export WANDB_MODE=offline

python tools/release/automoma-30k/convert_chunks_to_robotwin_dp3_zarr.py \
  --record-root "$ARCHIVE_ROOT/record" \
  --object-name microwave_7221 \
  --selection-manifest "$ARCHIVE_ROOT/manifests/selection_manifest.json" \
  --task-name microwave_7221 \
  --task-config automoma_30k \

cd /home/xinhai/projects/lerobot-arena/third_party/RoboTwin/policy/DP3
bash scripts/train_policy.sh robot_dp3 microwave_7221 microwave_7221_30k_local_trial 30000 train 42 0 "$TRAIN_OUT" \
  task.dataset.zarr_path="$ZARR" \
  training.num_epochs=300 \
  training.checkpoint_every=25 \
  training.val_every=25 \
  training.max_train_steps=100 \
  training.max_val_steps=10 \
  dataloader.batch_size=64 \
  val_dataloader.batch_size=64 \
  dataloader.num_workers=4 \
  val_dataloader.num_workers=4 \
  logging.mode=offline \
  checkpoint.save_ckpt=True
```
